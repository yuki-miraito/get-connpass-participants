# get-connpass-participants

## About

Connpassのイベントから参加者をマークダウンのリスト形式で取得するスクリプトです。

## 使い方

```bash
$ python opt/get-participants.py <コンパスのイベントページURL>
```

### 例

```bash
$ python opt/get-participants.py https://phper-oop.connpass.com/event/171008/
```
